# TaskForge

A decentralized task management system built on Stacks blockchain using Clarity smart contracts.

## Features
- Create and manage tasks
- Assign tasks to team members
- Update task status
- Task completion verification
- Task history tracking

## Contract Functions
- create-task
- assign-task 
- update-task-status
- complete-task
- get-task-details
- get-user-tasks
